What's New since the Beta 9

Recent Changes

I am unable to list everything. But here's what I remember:

  • Optimized profiles tab
  • We can now edit a profile
  • We can now add a profile to our favorites (a favorite profile will be displayed at the top of the profiles list)
  • We can now remove and edit a profile just from the profiles list
  • New Profile Format Version (1.1.0) that allow the user to select the map place in the load order
  • Unspecified iwad/source port in a profile now count as compatible with everything (Profiles with unspecified source port won't be displayed in the personal library)
  • IDGames mirrors
  • Recently played games in the personal library when no wad is selected
  • The Coordinator Server does not close game port anymore before the game launch
  • Fixed startup system by adding corresponding binaries
  • And some refactor, code upgrade, fixes and so much more!
